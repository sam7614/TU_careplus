## 🎯 위기 학생 판정 기준

시스템은 다음 조건 중 **하나라도** 만족하는 학생을 위기 학생으로 분류합니다:

1. **출석률 부족**: 평균 출석률 70% 미만
2. **학점 부족**: 직전학기 평점 2.0 미만  
3. **등록금 문제**: 등록금 상태가 '미납' 또는 '부분납'

## 🚀 실행 방법

### Windows 사용자
```cmd
# 간단 실행
run_app.bat

# 수동 실행
pip install -r requirements.txt
streamlit run app.py
```

### macOS/Linux 사용자
```bash
# 간단 실행
./run_app.sh

# 수동 실행
pip3 install -r requirements.txt
python3 -m streamlit run app.py
```
