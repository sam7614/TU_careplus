# 🎓 동명대학교 위기 학생 관리 시스템

Streamlit 기반의 위기 학생 식별 및 관리 대시보드

## 📋 프로젝트 개요

이 시스템은 학생 데이터를 분석하여 학업적 또는 재정적 위기에 처한 학생들을 자동으로 식별하고 관리할 수 있는 웹 대시보드입니다.

## 🎯 주요 기능

### 1. 상단 요약 메트릭
- 전체 학생 수
- 위기 학생 수 (빨간색 강조)
- 전체 평균 출석률

### 2. 위기 학생 자동 식별
다음 조건 중 **하나라도** 해당되면 위기 학생으로 분류:
- `평균_출석률` 70% 미만
- `직전학기_평점` 2.0 미만  
- `등록금_납부_상태`가 '미납'

### 3. 데이터 조회 및 필터링
- 위기 학생 명단 우선 표시
- 학과별 필터링 기능
- 전체 학생 데이터 조회 (확장 가능)

## 🛠️ 기술 스택

- **Python 3.7+**
- **Streamlit**: 웹 대시보드 프레임워크
- **Pandas**: 데이터 분석 및 처리
- **openpyxl**: Excel 파일 지원

## 📦 설치 및 실행

### 1. Python 설치
Python 3.7 이상이 필요합니다. [Python 공식 사이트](https://www.python.org/downloads/)에서 다운로드하세요.

### 2. 의존성 설치
```bash
pip install -r requirements.txt
```

### 3. 데이터 파일 준비
- `care_student.csv` 파일을 프로젝트 폴더에 위치시키세요
- 파일은 UTF-8 또는 CP949 인코딩이어야 합니다

### 4. 애플리케이션 실행
```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501`로 접속하세요.

## 📊 데이터 형식

CSV 파일은 다음 컬럼들을 포함해야 합니다:

| 컬럼명 | 설명 | 데이터 타입 |
|--------|------|-------------|
| 학번 | 학생 번호 | 문자열 |
| 이름 | 학생 이름 | 문자열 |
| 학과 | 소속 학과 | 문자열 |
| 학년 | 학년 | 숫자 |
| 직전학기_평점 | 이전 학기 GPA | 숫자 (0.0-4.5) |
| 평균_출석률 | 출석률 백분율 | 숫자 (0-100) |
| 현재_성적 | 현재 성적 | 숫자 |
| 상담_받은_횟수 | 상담 횟수 | 숫자 |
| 장학금_신청 | 장학금 신청 여부 | 문자열 (O/X) |
| 현재_평점 | 현재 GPA | 숫자 |
| 도서관_이용_횟수 | 도서관 이용 횟수 | 숫자 |
| 등록금_납부_상태 | 등록금 납부 상태 | 문자열 (완납/미납/부분납) |

## 🚨 위기 학생 판정 기준

학생이 다음 조건 중 **하나라도** 만족하면 위기 학생으로 분류됩니다:

1. **출석률 부족**: 평균 출석률이 70% 미만
2. **학점 부족**: 직전학기 평점이 2.0 미만
3. **등록금 미납**: 등록금 납부 상태가 '미납'

## 🎨 UI/UX 특징

- **직관적인 레이아웃**: 메트릭 → 위기학생 → 필터링 순서
- **반응형 디자인**: 다양한 화면 크기 지원
- **한국어 완전 지원**: 모든 UI가 한국어로 제공
- **시각적 강조**: 위기 학생 수를 빨간색으로 강조
- **상세 정보**: 각 위기 학생의 구체적인 위기 요인 표시

## 🔧 문제 해결

### 인코딩 오류
- CSV 파일을 UTF-8 또는 CP949 인코딩으로 저장하세요
- Excel에서 저장할 때 "CSV UTF-8" 형식을 선택하세요

### 파일을 찾을 수 없음
- `care_student.csv` 파일이 `app.py`와 같은 폴더에 있는지 확인하세요
- 파일명의 대소문자를 정확히 맞춰주세요

### 데이터 형식 오류
- 숫자 컬럼에 텍스트가 포함되어 있지 않은지 확인하세요
- 필수 컬럼이 모두 포함되어 있는지 확인하세요
